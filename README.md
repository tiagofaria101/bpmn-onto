# bpmn-onto

Convert BPMN 2.0 process diagrams into validated ISA-95 ontologies using LLMs with a verifier-agent pattern.

> 🚧 In active development. Not yet usable.

## Inspiration

This project is a clean-room reimplementation of ideas from the public OntoEKG paper. See `docs/design.md` (coming soon).
